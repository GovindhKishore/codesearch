# codesearch

> Work in progress 

A hybrid semantic, lexical, and structural code search CLI for Python codebases.

## What it does

Search any Python codebase using natural language queries from your terminal - combining three retrieval signals:

- **Lexical (BM25)** - fast, keyword-based search
- **Semantic (embeddings)** - handles vocabulary mismatch
- **Structural (call graph)** - surfaces architecturally relevant functions, not just textually similar ones

## Status

Currently under active development. Not ready for use yet.
